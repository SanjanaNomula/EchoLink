# Code of Conduct

## Our Pledge

We are committed to providing a welcoming and inclusive environment for everyone.

## Expected Behavior

- Be respectful
- Be constructive
- Be professional
- Help others learn

## Unacceptable Behavior

- Harassment
- Personal attacks
- Discrimination
- Trolling

## Enforcement

Project maintainers may remove content that violates this code of conduct.

## Scope

This code of conduct applies to all project spaces including issues, pull requests, and discussions.
