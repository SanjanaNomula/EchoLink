# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

## Supported Versions

| Version | Supported |
|----------|----------|
| 1.x | Yes |
| 0.x | No |

## Disclosure Process

1. Report the issue privately.
2. Allow maintainers time to investigate.
3. A fix will be released if necessary.
4. Public disclosure will follow after a patch is available.
