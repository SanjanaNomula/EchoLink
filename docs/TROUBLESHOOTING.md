# Troubleshooting Guide

## Microphone Not Working

### Possible Causes

* Microphone permissions denied
* Browser restrictions
* Device not connected

### Solution

1. Check browser permissions.
2. Verify microphone access.
3. Refresh the page and reconnect.

---

## Unable to Connect

### Possible Causes

* Invalid Peer ID
* Network issues
* PeerJS signaling problems

### Solution

1. Verify the Peer ID.
2. Check internet connectivity.
3. Try reconnecting.

---

## No Audio During Call

### Possible Causes

* Muted microphone
* Audio output device issue
* Browser autoplay restrictions

### Solution

1. Verify microphone is enabled.
2. Check speaker volume.
3. Allow audio playback in browser settings.

---

## Frequent Disconnections

### Possible Causes

* Unstable network
* Browser resource limits

### Solution

1. Use a stable internet connection.
2. Close unnecessary browser tabs.
3. Reconnect to the session.

---

## Reporting Issues

If the problem persists, create an issue using the provided GitHub issue templates.
