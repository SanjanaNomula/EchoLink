# Deployment Guide

## Overview

EchoLink can be deployed as a static web application.

## GitHub Pages

### Steps

1. Push your project to GitHub.
2. Open repository settings.
3. Navigate to Pages.
4. Select the deployment branch.
5. Save changes.

Your project will be available through GitHub Pages.

## Local Hosting

You can host EchoLink locally using:

```bash
python -m http.server 8000
```

Open:

```text
http://localhost:8000
```

## Production Hosting

EchoLink can be deployed using:

* GitHub Pages
* Netlify
* Vercel
* Firebase Hosting

## Requirements

* Modern browser
* HTTPS recommended
* WebRTC support enabled
